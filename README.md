# BDIA-Safety Walk

세이프티 워크는 시각 장애인을 위한 혁신적인 스마트폰 앱으로 안전하고 원활하게 걸을 수 있도록 도와줍니다. 앱은 실시간 음성 신호를 제공하여 횡단보도를 건널 때 사용자가 안전하게 걸을 수 있도록 도와줍니다. GPS를 비롯한 센서 데이터를 활용해 사용자의 현재 위치와 주변을 파악하고, 최적의 횡단보도 경로를 안내합니다. 또한 공공기관의 데이터를 활용하여 보행자 신호의 위치를 제공하고 사용자에게 최신 교통 상황을 제공합니다. 세이프티 워크는 시각 장애인의 독립적인 이동을 촉진하고 교통 사고를 예방하며 사회적 포용성을 높이는 것을 목표로 합니다.
